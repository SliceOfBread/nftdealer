Miscellaneous

Rule differences versus The Gallerist:

  • There is an option to play with more than 4 contracts on display (and I suggest using 8). When using this option, there is no option to 'deal more contracts'. What you see is what you can get. If you choose one of the contracts in the four leftmost locations, one contract will be added on the right (everything else moved left) when refilled. If you choose one of the other contracts, all four contracts on the left are discarded, everything else is then moved left and all spaces refilled.
  • In the case where the player cannot perform any action, the player will move to a location and a brown ticket is discarded. If there are no more brown tickets, then a different color is discarded (the color does not matter). If there are no more tickets of any color, then a random visitor is drawn from the bag and move to the plaza. This rule change assures that the game will end eventually.
